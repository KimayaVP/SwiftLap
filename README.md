# SwiftLapLogic

Real-time swim performance tracking with AI-powered feedback and race-day strategy.

## What It Does
SwiftLapLogic helps swimmers and coaches track progress, analyze performance, and hit race goals. Upload videos for stroke feedback, get personalized workouts when falling behind goals, and generate race-day warm-up plans.

## Who It's For
- Competitive swimmers tracking monthly/yearly goals
- Coaches managing multiple swimmer profiles
- Athletes wanting actionable feedback, not just raw times

## The Problem
Swimmers get timing data but don't know what to do with it. Coaches can't watch every stroke in group classes. Existing apps show numbers without explaining how to improve.

## Tech Stack
- **Frontend:** React Native (mobile app)
- **Backend:** Node.js + Express
- **Database:** PostgreSQL (swimmer data & progress)
- **AI/Analysis:** Python (video feedback)
- **API:** REST API linking coach and swimmer accounts

## 4-Week MVP Goals
- Log swim times and track monthly goals
- Upload videos and receive basic stroke feedback
- Coach dashboard to view multiple swimmers
- Predictive goal tracking with workout recommendations

## Getting Started
1. Clone this repository
2. Install dependencies: npm install
3. Set up PostgreSQL database
4. Configure environment variables in .env
5. Run development server: npm run dev
